# 使用typescript重构axios

## Features
- 在浏览器端使用xmlHttpRequest对象通讯
- 支持promise api
- 支持请求和响应的拦截器
- 支持请求数据和响应数据的转换
- 支持请求的取消
- JSON数据的自动转换
- 客户端防止xsrf

## 脚手架
git clone https://github.com/alexjoverm/typescript-library-starter.git axios
cd axios
- typedoc             自动生成文档并且部署到github pages
- jest                单元测试
- rollup              打包工具
- commitizen          生成规范化的提交注释
- semantic release    版本管理和发布
- husky               git hooks工具
